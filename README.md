# Homepage

## Description
This is the repository for my homepage.

## Installation
```bash
# Clone the repository
git clone https://github.com/xandll/homepage.git

# Change to project directory
cd homepage

# Install dependencies
npm install
```

## Usage
```bash
# Start development server
npm run dev
```
